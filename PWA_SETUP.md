## How to Add Your Custom Jumia Logo 🎨

### Option 1: Provide PNG Logo Files (Recommended)

1. **Prepare your Jumia logo** in PNG format with transparent background
2. **Create the following sizes** and place them in `/public/`:
   ```
   icon-192x192.png          (192x192 pixels)
   icon-512x512.png          (512x512 pixels)
   icon-maskable-192x192.png (192x192 pixels with 20% safe zone)
   icon-maskable-512x512.png (512x512 pixels with 20% safe zone)
   ```

3. **Maskable icons** should have important content in the center 80% area (safe zone)

### Option 2: Use ImageMagick to Auto-Generate

If you have a single logo image:

```bash
# Install ImageMagick
sudo apt-get install imagemagick

# Run the generation script
./scripts/generate-icons.sh
```

### Option 3: Use Online Tool

1. Visit https://realfavicongenerator.net/
2. Upload your Gadget World logo
3. Download the generated icons
4. Place them in `/public/` folder
5. Icons needed:
   - `icon-192x192.png`
   - `icon-512x512.png`
   - `icon-maskable-192x192.png`
   - `icon-maskable-512x512.png`

### Option 4: Replace SVG Icons

Simply replace these files with your logo SVG:
- `/app/icon.svg` (192x192 viewBox)
- `/app/apple-icon.svg` (180x180 viewBox)
